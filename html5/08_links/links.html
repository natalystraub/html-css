Links Internos

Para adicionar links internos dentro do HTML, você precisa informar o seguinte comando; 

          << a >  href="../01_paragrafos/paragrafos.html" rel="next" >Paragrafos << / a >

Exemplo de código

Podemos também adicionar um link para retornar a página anterior
