Apreendendo sobre HTML

O que é o HTML?

HTML (HyperText Markup Language ou Linguagem de Marcação de Hipertexto) é a base estrutural de todas as páginas da web, definindo elementos como títulos, parágrafos, imagens e links através de tags.

Curiosidades

HTML foi criado em 1991.

O que é tags semânticas?

Tags semânticas HTML são elementos que definem claramente o significado do seu conteúdo para navegadores e desenvolvedores, indo além da simples aparência visual.
